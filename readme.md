# PlantPath 875: Phylogenetic Practicum

A 1-credit course in the practice of phylogenetic inference from DNA sequence data (UW-Madison). Students will practice all the steps of state-of-the-art phylogenomic analyses with hands-on weekly activities.

## Learning outcomes

By the end of the course, you will be able to
1. Reproduce all the steps of a phylogenomics data analysis of a published paper
2. Learn best computing and reproducibility practices for phylogenomic data analyses
